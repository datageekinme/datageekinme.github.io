## Data Structures in Python:

### Lists:
**Ordered sequence of values of other data types like int, boolean or string**

```python
# Example:
[1,2,3] or ['a','b','c']
```

### Dictionaries:
**A collection of key: values**

```python
# Example:
{"id": 1, "first_name": "Keith", "last_name": "Mascarenhas", "salary": "80000.00"}
```
